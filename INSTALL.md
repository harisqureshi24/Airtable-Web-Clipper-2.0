# Simple Installation Guide

## Install This Extension in Airtable

### Option 1: Remix from GitHub (Easiest)

1. **Go to your Airtable base** (in your browser)

2. **Click "Extensions"** (top right corner)

3. **Click "Add an extension"**

4. **Click "Build a custom extension"**

5. **Click "Remix from GitHub"**

6. **Paste your GitHub repo URL**:
   ```
   https://github.com/YOUR_USERNAME/airtable-enhanced-web-clipper
   ```

7. **Done!** The extension will install automatically

---

### Option 2: Copy & Paste Code

If "Remix from GitHub" doesn't work:

1. Open your Airtable base
2. Click "Extensions" → "Add an extension" → "Build a custom extension"
3. You'll see a code editor
4. Copy all files from this GitHub repo into the editor
5. The extension will start running

---

## What This Extension Does

✅ **Add Date Fields** - The native web clipper doesn't support dates
✅ **Create Linked Records** - Create new records in related tables inline
✅ **All Field Types** - Full support for text, numbers, selects, ratings, etc.

---

## Files Overview

- `frontend/index.tsx` - Main extension code
- `block.json` - Extension configuration
- `package.json` - Dependencies
- `.block/remote.json` - Airtable connection config
